# TF-100: Terraform Fundamentals Lab

A comprehensive, hands-on Instruqt lab teaching Terraform fundamentals from zero to proficient.

## 🎯 What You'll Learn

By completing this lab, you will be able to:

- Write and execute basic Terraform configurations
- Use variables, loops, and functions effectively
- Provision real infrastructure (VMs, networks) using Libvirt
- Manage Terraform state and use CLI commands confidently
- Debug Terraform issues independently
- Apply Terraform best practices

## 📋 Prerequisites

- **Theoretical Knowledge**: Understanding of Infrastructure as Code concepts (why IaC > click-ops)
- **Terraform Experience**: None required - this lab assumes zero Terraform knowledge
- **Time Commitment**: ~8 hours (completable in one working day)

## 🏗️ Lab Structure

### Challenge 1: Introduction to IaC & Terraform Basics (1.5 hours)
Learn Terraform fundamentals, HCL syntax, and basic resource management.

### Challenge 2: Variables, Loops & Functions (1.5 hours)
Master variables, for_each, count, and built-in functions.

### Challenge 3: Infrastructure Resources (2 hours)
Provision real VMs and networks using Libvirt.

### Challenge 4: State Management & CLI (1.5 hours)
Understand state files, CLI commands, and debugging techniques.

### Challenge 5: Skills Assessment (1.5 hours)
Test your skills with a real-world scenario (no step-by-step instructions).

1. **Launch the lab** - Environment is pre-configured and ready
2. **Use both interfaces**:

   - **Code Editor**: View and modify configuration files

4. **Validate your work** - Check scripts confirm you're on track
5. **Complete the assessment** - Test your skills independently

## 💡 Tips for Success

1. **Read carefully** - Instructions include everything you need
2. **Use the terminal** - Practice typing commands (don't just copy-paste)
3. **Experiment** - Try variations to deepen understanding
4. **Use terraform console** - Great for testing expressions
5. **Read error messages** - They usually tell you what's wrong
6. **Take breaks** - Better to learn well than rush through
